# Gemini MCP Setup

This repository demonstrates the successful setup and run of the **Gemini CLI** using a **GitHub MCP token**.

---

## Overview

The purpose of this project is to verify that Gemini CLI can:

- Connect to GitHub MCP using a personal access token.
- Read cached credentials and previous session memory.
- Successfully execute commands in debug mode.
